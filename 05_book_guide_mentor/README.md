# 독서 가이드 멘토 챗봇

10년 이상 경력의 독서 큐레이터 멘토 역할을 하는 CLI 챗봇입니다.
5단계 설문을 통해 사용자 프로필을 파악하고, 맞춤형 도서 추천과 독서 로드맵을 제공합니다.
OpenAI gpt-4o를 활용합니다.

---

## 설치 및 실행

### 1. 의존성 설치

```
pip install -r requirements.txt
```

### 2. 환경 변수 설정

프로젝트 폴더 안에 `.env` 파일을 생성합니다.

```
OPENAI_API_KEY=sk-...
```

- **OPENAI_API_KEY**: 필수. [OpenAI 플랫폼](https://platform.openai.com/)에서 발급

### 3. 실행

```
python main.py
```

---

## 사용법

프로그램을 실행하면 멘토가 먼저 인사를 건네고, 아래 5단계 설문을 순서대로 진행합니다.

### 설문 진행 순서

| 단계 | 질문 내용 |
|---|---|
| 1 | 관심 주제 / 장르 |
| 2 | 해당 분야 지식 수준 (입문 / 중급 / 고급) |
| 3 | 선호 형식 (얇은 책 / 두꺼운 책 / 이론 / 실용 / 에세이) |
| 4 | 독서 목적 (취미 / 업무 / 자기계발 / 문제 해결) |
| 5 | 이전에 읽은 책 (없으면 "없음") |

### 추천 결과 형식

설문 완료 후 다음 내용을 출력합니다.

- **독자 프로필 요약**: 수집된 정보 정리
- **추천 도서 목록 (3~5권)**: 제목, 저자, 추천 이유, 난이도, 예상 독서 시간
- **독서 로드맵**: 추천 순서 및 단계별 안내
- **추가 팁**: 독서 효과를 높이기 위한 조언

### 대화 예시

```
[멘토] 안녕하세요! 저는 독서 큐레이터 멘토입니다.
       어떤 주제나 분야의 책에 관심이 있으신가요?

[나] 심리학이요

[멘토] 좋은 선택이에요! 심리학은 정말 흥미로운 분야죠.
       해당 분야에 대한 본인의 지식 수준은 어느 정도인가요?
       (입문 / 중급 / 고급)

[나] 입문
...
```
