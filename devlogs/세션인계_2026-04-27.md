## 중요 개념 정리

### 로컬 eval vs Docker — ChromaDB 별개
- 로컬 eval → `08_FitStep_API/data/chroma_db/` (로컬 폴더)
- Docker → `chroma_data` named volume (Docker 내부)
- 둘은 완전히 독립적 → 각각 따로 인덱싱 필요

### 인덱싱 비용
- GPT-4o-mini로 1,958번 API 호출 (체력측정 950 + 운동추천 1008)
- 예상 비용: 약 $0.55 (OpenAI 계정에서 차감)
- **한 번만** 실행하면 이후 재사용 가능

### eval 지표
| 지표 | 설명 |
|------|------|
| Precision@3 | 검색된 상위 3개 청크의 관련성 |
| Faithfulness | 답변이 검색 컨텍스트에 근거하는 정도 |
| Requirement Coverage | 필수 키워드 포함 여부 |
| Rule Check | 운동 처방 규칙 준수 여부 |
